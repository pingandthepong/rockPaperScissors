## 📱 Break Points

| 구분                    | 기기 예시            | 가로 해상도 기준              |
| ----------------------- | -------------------- | ----------------------------- |
| **Small (모바일)**      | 스마트폰 세로        | `max-width: 480px` or `576px` |
| **Medium (태블릿)**     | iPad 세로/가로       | `max-width: 768px`            |
| **Large (노트북)**      | 작은 노트북, 일반 PC | `max-width: 1024px`           |
| **XL (데스크탑)**       | FHD 모니터           | `max-width: 1280px`           |
| **2XL (와이드 모니터)** | QHD, 4K              | `max-width: 1536px`           |
